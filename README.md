This repository contains an analysis of the **World Risk Index**, which assesses the disaster risk for countries worldwide, considering factors such as vulnerability, exposure to natural hazards, and the ability to cope with and adapt to these risks. The analysis is done using in a jupyter notebook.


The dataset consists of 11 years data of multiple countries with features including:

- Region
- WRI
- Exposure
- Vulnerability
- Susceptibility
- Lack of Coping Capabilities
- Lack of Adaptive Capabilities
- Year
- WRI Category
- Exposure Category
- Vulnerability Category
- Susceptibility Category
- Feature Name	

Link to dataset: https://www.kaggle.com/datasets/tr1gg3rtrash/global-disaster-risk-index-time-series-dataset?resource=download
